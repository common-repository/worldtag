Use this plugin to link keywords in your blogs to other blog entries about the same subject.

== Description ==

This reads posts from your blog, and identifies the wt: identifier, tuning them into links. The links go back to a database of pages that have used the same link. So you can collectivly see who is blogging about the same subject.

For example, if you are writing about a new release of wordpress, you could write wt:wpthreepointtwo. This would link to a page that has a list of other blog entries about the same subject.

Please understand that this plugin stores links to your blog in my databae. No personel information is stored, jsut a link to the blog, the host site, keyword used and the date it was entered.

== Installation ==

1. Upload `WorldTag.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
